# Система управления задачами и мониторинга эффективности

**Описание**:  
Это веб-приложение, разработанное на Python с использованием Flask, предназначено для эффективного управления проектными задачами и мониторинга производительности команды.

## 🚀 Основные возможности

- **Управление задачами**: создание, редактирование, удаление и отслеживание статуса задач.
- **Аналитика производительности**: визуализация данных о выполнении задач и эффективности сотрудников.
- **Управление проектами**: создание и управление проектами, назначение задач участникам.
- **Аутентификация пользователей**: система регистрации и входа для разных ролей пользователей.

## 🛠 Технологический стек

- **Backend**: Python 3.x, Flask
- **Frontend**: HTML, CSS, JavaScript
- **База данных**: SQLite (по умолчанию), поддержка PostgreSQL
- **Библиотеки**: Flask-Login, SQLAlchemy, Matplotlib для визуализации данных

## 📁 Структура проекта

- `app.py` — основной файл приложения
- `templates/` — HTML-шаблоны
- `static/` — статические файлы (CSS, JavaScript)
- `models.py` — модели базы данных
- `charts.py` — скрипты для генерации графиков и диаграмм


## 📌 Примечания

- Приложение предназначено для использования в небольших командах и проектах.
- Для развертывания в производственной среде рекомендуется настроить PostgreSQL и использовать WSGI-сервер, например, Gunicorn.
